CORE SETUP
===========
1. Install Python 3.11
	a. Add in system path.	 
2. Install Ollama and a Model of choice
3. Install open-webui "pip install open-webui"

AERISCORE SETUP
===============
1. Create a subfolder for AERISCORE
	a. c:\myCoolAI

2. Rename "personality.txt.dist" -> "personality.txt"
	a. Edit to your liking.
	b. for ideas visit https://characterhub.org
3. Rename "info.cfg.dist" -> "info.cfg"	
	line #1: path to this folder (ex. c:\mycoolai)
	line #2: PATH to your open-webui
	line #3: Companion Name.
	line #4: Your Name
	
4. ai_run.bat
5. open browser to localhost:8080 to chat with your AI
** WARNING **
If you do not see the AERISCORE logo (instead of OPEN-WEBUI) the mod did not 
get injected, and I am aware of it. Keep checking back.!!
